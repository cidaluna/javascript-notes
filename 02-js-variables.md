# JS Variables

- **var** 
- **let**
- **const** (_value can NOT be updated_)

Keyword             |  var   | let   | const 
--------------------|--------|-------|------
Global Scope        |  ✅   |  ❌   |  ❌
Function Scope      |  ✅   |  ✅   |  ✅
Block Scope         |  ❌   |  ✅   |  ✅
Can Be Reassigned?  |  ✅   |  ✅   |  ❌
Can Be Redeclared?  |  ✅   |  ❌   |  ❌
Can Be Hoisted?     |  ✅   |  ❌   |  ❌


| Legenda     |
|-------------|
| ✅ - Yes    |
| ❌ - No     |


```js
var name = "Cida";
let age = 36;
const cool = true;
```
